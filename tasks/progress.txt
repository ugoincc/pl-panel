## Codebase Patterns
- styles.css uses oklch() color space for shadcn CSS variables; match this format when adding/updating variables
- PulseLab-specific variables (--sea-ink, --lagoon, --palm, --pl-navy, etc.) live alongside shadcn variables in styles.css — do NOT touch them
- TypeScript check: use `npx tsc --noEmit` (no "typecheck" script in package.json); pre-existing Nav.tsx unused-import errors are expected and not blocking
- Available quality scripts: `npm run lint` (biome), `npm run check` (biome), `npx tsc --noEmit` (types), `npm test` (vitest)
- The project is on branch `ralph/shadcn-panel-refactor`
